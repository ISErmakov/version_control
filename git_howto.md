# How to work with git

## Инициализация репозитория

```sh
git init
```

## Проверка текущего состояния

```sh
git status
```
## Сохранение версии

```sh
git commit
```
## Работа с версиями/Переключение между ветками

```sh
git checkout
```

## Работа с ветками

```sh
git branch
```

## Объединение веток

```sh
git merge
```
